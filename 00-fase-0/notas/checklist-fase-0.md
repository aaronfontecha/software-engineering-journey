# Checklist — Fase 0

## Objetivo de la fase

Preparar mi entorno técnico para estudiar Software Engineering de forma estructurada.

## Habilidades practicadas

- Crear y organizar carpetas de estudio.
- Abrir proyectos en Visual Studio Code.
- Usar la terminal integrada.
- Navegar carpetas con `cd`.
- Listar contenido con `ls`.
- Ver mi ubicación actual con `pwd`.
- Crear carpetas con `mkdir`.
- Crear archivos con `New-Item`.
- Limpiar la terminal con `clear` o `cls`.
- Instalar Git.
- Configurar nombre y correo en Git.
- Inicializar un repositorio con `git init`.
- Revisar cambios con `git status`.
- Preparar cambios con `git add .`.
- Guardar versiones con `git commit`.
- Ver historial con `git log`.
- Conectar el repositorio local con GitHub.
- Subir cambios con `git push`.
- Usar `.gitkeep` para conservar carpetas vacías.
- Crear un `README.md` básico.
